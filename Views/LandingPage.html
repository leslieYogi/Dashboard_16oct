As you are aware, the Federal Perkins Loan Program will expire on September 30, 2017. There are two critical items that you need to be aware of effective October 1, 2017. 1) As of 12:01 AM, Sunday, October, 1, 2017, Heartland ECSI will cancel all unsigned Perkins Loan Promissory Notes on our online signing platform. Please note that we will still accept advance files for Perkins Loans. By submitting an advance file to Heartland ECSI, you attest that all Perkins loans within the advance file are valid and were first disbursed prior to October 1, 2017 in conformance with the Federal Perkins Loan Program Extension Act of 2015, enacted December 18, 2015.
